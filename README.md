# 🚢 Task 1: Data Cleaning & Preprocessing | Titanic Dataset

> **Machine Learning Internship Task 1**
> Transforming raw data into a clean, ML-ready dataset using Python and Google Colab.

---

## 📌 Project Overview

This project demonstrates the complete **Data Cleaning & Preprocessing** pipeline on the **Titanic Dataset**. The goal is to improve data quality by handling missing values, encoding categorical features, scaling numerical data, and removing outliers before applying machine learning algorithms.

---

## 🎯 Objective

✔️ Explore the dataset
✔️ Handle missing values
✔️ Encode categorical features
✔️ Standardize numerical features
✔️ Detect & remove outliers
✔️ Generate a clean dataset for Machine Learning

---

## 🛠️ Tech Stack

* 🐍 Python
* 📒 Google Colab
* 🐼 Pandas
* 🔢 NumPy
* 📊 Matplotlib
* 🎨 Seaborn
* 🤖 Scikit-learn

---

## 📂 Dataset

**Dataset:** Titanic Dataset

The dataset contains passenger information such as age, gender, ticket class, fare, cabin details, embarkation point, and survival status.

---

## ⚙️ Project Workflow

* 📥 Imported and explored the dataset
* 🔍 Checked data types and missing values
* 🧹 Filled missing values using median and mode
* ❌ Removed unnecessary columns
* 🔤 Encoded categorical variables using Label Encoding
* 📈 Standardized numerical features using StandardScaler
* 📦 Visualized outliers with boxplots
* 🚀 Removed outliers using the IQR method
* 💾 Saved the cleaned dataset
